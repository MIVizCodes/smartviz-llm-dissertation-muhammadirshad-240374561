# LLM-Powered Smart Visualization for Building Monitoring (Dissertation)

This repository contains all code, data samples, figures and the interactive Streamlit app accompanying the dissertation.

## Structure
- `llm_project_smartviz_M_Irshad.ipynb`: Main Jupyter notebook with LLM integration, anomaly detection, Multi-Agents, results, observations, plots.
- `app.py`: Streamlit dashboard for interactive model experience (run queries like "hottest room right now").
- `data/`: Sample CSV files (building sensor data).
- `figures/`: All graphs, heatmaps, bar plots generated in the notebook.
